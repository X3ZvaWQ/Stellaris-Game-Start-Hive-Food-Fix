## Mod 简介

> 蠢驴小时候妈妈给他买了一艘合金船，从此以后再也不给没让他吃上过饭，一直到上天都是0食物

### 概述

不知道什么时候开始蜂巢如果选择合金船开局就是0食物，虽然也没什么大的影响，但是玩起来浑身难受，本mod负责开局补发缺少的200食物。

### 兼容性

- 仅适用于开局的存档
- 仅对选择了合金船的蜂巢思维帝国生效
